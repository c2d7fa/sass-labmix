    contrast

    @function contrast($bg, $fg) { ... }

    Description

    Calculate the contrast between two colors.

    Note that the foreground/background distinction is only relevant when dealing with semi-transparent colors. In that case, transparency is interpreted with the worst case.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $bg

    background color

    		Color—none
    $fg

    foreground color

    		Color—none

    Returns

    Number —

    between 1 and 21

    Requires

    • [function] luma

    Used by

    • [function] contrast-color
    • [function] contrast-check
